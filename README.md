# Supermarket_sales_analysis
## Description
End-to-end data analysis project involving data cleaning, exploration, and visualization of supermarket sales data to drive business insights and decision-making.

## Project overview
This project analyzes supermarket sales data to uncover insights into customer behavior, product performance, and revenue trends

## Objectives
i.Top-selling products
ii.Revenue trends
iii.Quantity Sold
iv.Payment preferences

## Tools Used
power point and excel

## Key Insights
i.Total revenue 
ii.Total Qunatity Sold
iii.Top location by Revenue
